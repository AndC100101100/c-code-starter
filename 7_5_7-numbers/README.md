## integers
an unsigned integer range has positive numbers only, and a signed range has an equal amount of positive and negative numbers.

## float
a floating point value is a number with a decimal point, the most common in C are float(32 bit single precision floating numbers that allow for 6 decimal digits precision) and double

## double

This is 'double' what a float can offer. It's a 64-bit data type and allows up to 15 decimal digits of precision
