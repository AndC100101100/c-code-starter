# c-code-starter
The following repo is being used to save some snippets regarding the C programming language. 

C is a compile language created in 1972 by the computer scientist Dennis Ritchie. Programs created in C can practically run on any system, it is also a portable language, meaning you can write it it, and compile it on any other system and have it work the same way. A few other characteristics include:
* Pretty fast
* A really mature language, with it most recent version being c23
* Cross-compilable
